# ios-swift-start
IOs Swift starting point 

## Conventional Commits

* **Install git hook to validate commits**
Requires NodeJs installed
```sh
> npm install
```

* **update changelog for current version**
```sh
# first release
> npm run release -- --first-release
# next releases
> npm run standard-version 
```
